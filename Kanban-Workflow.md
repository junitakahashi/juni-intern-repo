## How can you improve your workflow using Kanban principles?

I can improve my workflow by:
- Breaking tasks into smaller pieces so they move across the board more smoothly
- Moving cards immediately when status changes, keeping everything up to date
- Avoiding starting new tasks before finishing current ones
- Reviewing my board daily to stay aware of deadlines and workload
- Marking blocked tasks clearly so I can ask for help sooner
These habits help me stay organised, maintain clarity, and work more efficiently.
